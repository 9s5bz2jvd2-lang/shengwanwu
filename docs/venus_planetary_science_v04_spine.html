三、目前论文已报告 / 已观测之九条硬事实

序事实证据节点边界说明
1金星云层主成分为浓硫酸 H2SO4，质量分数约 75%–98%；水为次要成分。v1_003云层成分事实；可作为后续化学与宜居性讨论基底。
2上层云为光化学云：SO2 与 H2O 参与光化学反应形成 H2SO4 蒸气，随后凝结在凝结核上。v1_008说明上层云质量负载受光化学过程调节。
3云顶 H2SO4 光化学生成后，约 86% 向上输运并重新凝结入云滴，约 14% 向下扩散并热分解，形成闭合硫循环。v1_023定量模型结果；应用时须注明为模型报告。
4云层温压约 300–350 K、0.1–1 bar，但云滴主要为 H2SO4，水活度远低于已知极端微生物可承受范围。v1_038宜居性判断不可只看温压；水活度/浓硫酸是硬边界。
5金星云酸度观测存在显著不确定性：高度难确定、遥感窗口更敏感于云不透明度、硫酸光谱插值有非线性等。v1_004这是观测边界，适合升格为后续研究问题。
6Lefèvre 等小尺度湍流模型给云层 Kzz 约 106–108 cm²/s，高于多数旧大气化学输运模型；且 Kzz 依物种寿命而变。v1_016动力学与化学耦合关键事实；须注明为模型推导。
7未知 UV 吸收体与 0.32–0.5 μm 低反照率及云纹运动相关；320–400 nm 吸收增强不能由 SO2 完全解释；约贡献云顶太阳加热 50%。v1_026可作为行星科学枢纽问题：反照率、加热、云微物理与成分未明。
8UV 吸收体主要位于上云区 >57 km；赤道云较中高纬更暗；365 nm 反照率有强年代变化，丰度可变约 2 倍。v1_027提示空间差异与年代变化；不可直接判定吸收体成分。
920 种生物氨基酸中，19 种的氨基酸骨架在浓硫酸中保持完整；色氨酸完全反应降解。v5_001有机物稳定性实验支持；不能由此推出生命存在。

四、可由事实导出的候选方向

1. 云现象须跨域解释

金星云层现象不能只看化学；需联动动力学、辐射、云微物理、观测高度与时间变化。

2. 宜居性须守硬边界

温压虽较温和，但水活度与浓硫酸环境极端；宜居性讨论须先过此边界。

3. UV 吸收体是枢纽

它连接反照率、云顶加热、纬向差异、年代变化与云微物理，是行星科学主问题之一。

4. 有机物稳定性需谨慎表达

氨基酸骨架稳定性比旧直觉乐观；但仅能说明“有机物未必一概不稳”，不可推出生命结论。

The coupling between radiative balance, UV/IR signatures, and cloud microphysics is under-specified.

A planetary-scale coupling across the domains named in gap may explain the Venus cloud behavior better than treating each domain separately.
